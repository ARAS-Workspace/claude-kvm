# Phantom-WG Installation Task

Complete the following task on the XFCE desktop (1280x720):

1. Open Firefox from the bottom taskbar.
2. Navigate to www.phantom.tc and wait for the page to load.
3. Scroll down or page down to find the "QUICK INSTALL" section with the curl command.
4. Copy the install command: `curl -sSL https://install.phantom.tc | bash`
5. Open a terminal from the taskbar.
6. Paste the command and press Enter to start the installation.

## Important Notes

- Before scrolling, click on the page body to ensure it has focus.
- Use cursor_crop after clicking to confirm where the cursor landed.
- If the page has language buttons (TR/EN), click EN first to switch to English content.
- If scroll or pagedown doesn't change the page, try clicking a different area of the page body and retry.

- To paste in terminal use Ctrl+Shift+V (not Ctrl+V).
- Verify each major step before moving to the next one.
